CSS3 Transitions, Animations and Transformations

Transition VS Animation

In CSS, a transition is when the element state or property changes gradually over a specified amount of time. You can change the color of something when hovering over the element, or use JavaScript to adjust the size of an element on change as well. Transitions are less complex than animations; although, an animation will allow you more control over the changes being applied to the element.

Transitions

Some examples of transition properties are:

• transition : shorthand the following four properties [property, duration, timing-function, delay]

• transition-delay : defines when the transition begins

transition-delay: 1s;
• transition-duration : defines the length of the transition
transition-duration: 2s;
• transition-property : defines the name of the property that you're applying the transition to (You can pretty much use any property that accepts a numeric value i.e. width, height, color.. etc).
transition-property: width;
• transition-timing-function : (optional) defines the speed curve of the effect
transition-timing-function: ease-in-out;
To create a transition you need a CSS property that you want to add the transition to and how long you want the effect to occur. If a duration isn't specified, nothing will happen.

Transformations

The transformation property can be used to rotate, skew or scale an element, thus creating a 2D or 3D transformation.

/* Syntax */
div.transform {  
    transform: rotate(90deg);
    transform: skewY(30deg);
    transform: scaleY(1.5);
  }

JavaScript Events

JavaScript is able to "react" on HTML events. It allows you execute code when an event is detected.

This can be done using either single qoutations or double qoutations.



An example being:

element event='some JavaScript' or element event="some JavaScript"

Onclick Event

This event occurs when an element is clicked on.

Transition

This is an event and property. You can use the transitionend event to change the element once the transition completes. You can also use several transition properties to style the transition event.

The transition property is short for the following four properties:

transitionProperty, transitionDuration, transitionTimingFunction, and transitionDelay.

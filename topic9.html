JavaScript Events

Events

HTML events are things that happen to HTML elements. It can be something the user does themselves, or something the broser does. Javascript is able to react on the events. Event handlers make it so things happen when: a page loads, a button is clicked, the user inputs data, an object is moused over, you touch something on your mobile device, and many other things.

An Event would be something like the following:

• onchange - when an element has changed • onclick - when a user clicks an element • onload - when the browser has fully loaded • onmouseover - when the mouse is moved over an element Methods are also used in order to allow the JavaScript to work with the events. By using the addEventListener, event attributes can call JavaScript functions, execute JavaScript code directly and impact how the events are handled or sent, to name a few.

The syntax for using events is as follows:

• onclick="JavaScript"

• onclick="YourFunctionWithJavaScript()"

You can also use the addEventListener method which I referred to earlier which looks like this: 

// the "click" is the event and displayDate is the function
    document.getElementById("myBtn").addEventListener("click", displayDate);

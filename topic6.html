DOM Document Object Model

Tree of Objects

With the Document Object Model, Javascript can interact with it in multiple ways:

• change HTML elements

• change HTML attributes

• change CSS styles

• remove existing HTML elements and attributes

• add new HTML elements and attributes

• react to existing HTML events

• create new HTML events

createElement() Method

This method allows you to create an Element node with the specified name. Some examples would be creating a button or a paragraph element.

Syntax:

var elem = document.createElement(nodename);

Paragraph example:

Detailed syntax for creating an element:

function createBtn() {
// create the element (in this case, a button)
var btn = document.createElement("BUTTON");
// add text to the element
btn.innerHTML = "You made one!";
// find an existing element (in this case, 
// it's the element with the id "btn1")
var elmnt1 = document.getElementById("btn1");
// append the new element to the existing one
elmnt1.appendChild(btn);
}

insertBefore()

You can specify the location to append an element to. For example, insertBefore() allows you to place the element in front of any others.

Detailed syntax:

function insertParaBefore() {
// create a paragraph element
var para = document.createElement("p");
// create a text node
var node = document.createTextNode("I decided to add this before my first p element example.");
// append the text to the paragraph element we created
para.appendChild(node);
// find existing elements
var elmnt = document.getElementById("div1");
var child = document.getElementById("p2");
// insert our element before the other
elmnt.insertBefore(para,child);
}

remove()

This method is pretty straight forward and allows you to remove an existing element. 

// find your existing element
var elmnt = document.getElementById("btn1");
// remove it
elmnt.remove();

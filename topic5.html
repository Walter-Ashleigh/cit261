Local Storage

Web Storage API

Web storage API provides a way of storing data in a browser differently than when using cookies. The localStorage property has no expiration date and will remain stored in the browser even after it's been closed. It also has the highest max storage limit compared to sessionStorage (in which the data is only stored for a session) and cookies (which has the least amount of storage space).

Some things to remember:

• Keys and values in localStorage will always be strings.

• With a storage object, you are able to set, retrieve and remove data.

• The syntax for the local storage object: myStorage = window.localStorage;

A few ways to use the localStorage property:

// Set an item
localStorage.setItem("myName", "Ashleigh");
// Retrieve an item
var cat = localStorage.getItem('myName');
// Remove an item
localStorage.removeItem('myName');
// Clear all items
localStorage.clear();

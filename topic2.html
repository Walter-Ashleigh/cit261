JavaScript Objects

Objects and Properties

Objects are just like variables (which are containers for data values); although, and object can store many values. Each property is written as name:value pairs. For example, We can create an object named Ashleigh and assign it the values, female, 31, hazel, and brown. These values would be paired with the defined properties we come up with. For this example I have several properties and they're named: gender, age, eyes and hair. Just like a regular person, the values of each of these properties can change, but these defined ones are specific to me, Ashleigh. The syntax that goes along with this is as follows: 

// Create an object named Ashleigh:
var ashleigh = {gender:"female", age:31, eyes:"hazel", hair:"brown"};

CSS3 Transitions

Transition Basics & Speed Curve

With the transition property, you're able to specify the properties that will have the transition effect, the speed of the transition and the acceleration and/or deceleration.

The speed curves, which are the values of the transition-timing-function property, allow you many options to play with in deciding how you want the transition to look when it occurs.

• ease - slow start, then fast, then end slowly (this is default)

• linear - same speed from start to end

• ease-in - slow start

• ease-out - slow end

• ease-in-out - slow start and slow end

• cubic-bezier(n,n,n,n) - the most customizable

You can refer back to the transition/transformation and animation topic here for more information on the values of the transition property.

Examples of Transitions

Changing A Square To Circle: Uses the linear speed curve.

Hello
transition-property: all;
transition-duration: .5s;
transition-timing-function: linear;

Adding An Inset Border: Uses the ease speed curve.

transition-property: all;
transition-duration: 1s;
transition-timing-function: ease;

Using Cubic-Bezier: You can see a bounce in the speed curve.

transition-property: all;
transition-duration: 1s;
transition-timing-function: cubic-bezier(.23,1.07,.24,-0.79);

JSON

Parse

JSON allows you to exchange data to and from the server. When the data is recieved it's always a string. You can use JSON.parse() to parse the data into a JavaScript object.

We start with a variable, in this case I use the JSON format and create the txt variable. It contains keys and values which I will parse and display into a sentence.

var txt = '{"name":"Ashleigh", "sport":"football", "team":"Chargers", "year":"2018"}'

To parse we use this syntax:

var obj = JSON.parse(txt);

I'm now creating a new object called 'obj' where my txt data will be parsed into.

The last step is to get my element by id and properly format my sentence:

document.getElementById("display").innerHTML = obj.name 
+ " has always enjoyed " 
+ obj.sport
+ ". She really wishes the " 
+ obj.team 
+ " were doing as well as they did in the " 
+ obj.year
+ " season.";

Stringify

The data that's passed with JSON is formatted in JSON; However, we can use JSON.stringify() to convert that data to a string.

In this example we start with a JavaScript object:

var obj = { name: "Ashleigh", age: 31, city: "Melbourne", state: "Florida" };

To convert this data to a string we will use the syntax:

var myJSON = JSON.stringify(obj);

This creates a new variable called 'myJSON' which accepts my original JavaScript object into the parameters of the stringify function. Now that it's a string it can be sent to the server.

Here we get the element by id and use it to display our string:

document.getElementById("display2").innerHTML = myJSON;

CSS3 Using JavaScript

Using JavaScript to Change Styles

You can change different styles of an element using JavaScript. You can set the style of an element when you append a CSS property to the style property and then specify a value.

Syntax to change a style inline:

document.getElementById("demo").style.color = "pink";

Syntax when using events:

<button type="button"
onclick="document.getElementById('demo').style.color = 'white'">
Click Me!</button>;

You can also use the setAttribute() method to change styles but it's not recommended because this way can overwrite the CSS properties you have specified in your style attribute.

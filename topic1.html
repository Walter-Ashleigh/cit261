Loops

Loops will execute a block of code numerous times.

for: loops through a block of code a number of times

for/in: loops through properties of an object

for/of: loops through values of an iterable object

while: loops through a block of code while a specified condition is true

do/while: same as the while loop, just different syntax

Example of a for loop:

            var months = ["January", "February", "March", "April", "May", "June", "July",
                "August", "September", "October", "November", "December"];
                var i; 
                var len = months.length;
                var txt = "";
                for (i = 0; i < len; i++) {
                txt += months[i] + "<br>";
                }
                document.getElementById("monthp").innerHTML = txt;

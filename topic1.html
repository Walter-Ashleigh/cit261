JavaScript

Loops

Loops will execute a block of code numerous times.

• for: loops through a block of code a number of times

• for/in: loops through properties of an object

• for/of: loops through values of an iterable object

• while: loops through a block of code while a specified condition is true

• do/while: same as the while loop, just different syntax

Example of a for loop:

            var months = ["January", "February", "March", "April", "May", "June", "July",
                "August", "September", "October", "November", "December"];
                var i; 
                var len = months.length;
                var txt = "";
                for (i = 0; i < len; i++) {
                txt += months[i] + "<br>";
                }
                document.getElementById("monthp").innerHTML = txt;

Conditional Statements

Conditional statements allow you to perform a certain action based on a specified condition.

• if: the code would be executed if the condition is true

• else: the same code from above would be executed if the condition is false

• else/if: specifies a new condition to test if the first condition is false

• switch: specifies multiple different blocks of code to be executed

Example of an if/else statement:

            function getGreeting() {
                var hour = new Date().getHours(); 
                var greeting;
                if (hour < 18) {
                    greeting = "Hope you're enjoying your day!";
                } else {
                    greeting = "Hope you're enjoying your evening!";
                }
                document.getElementById("greet").innerHTML = greeting;
                }

Functions

A function is a block of code that performs a specific task and is executed when it's called. This can be done when the window is opened as well as other ways such as a click of a button. 

function myFunction() {
// code to be executed
}

Variables

Variables are containers that store data values.

var name = "Ashleigh";
var greeting = "Hello there!";
var x = 5;

Parameters

The function parameters are the names listed in the function definition.

function functionNames(parameter1, parameter2, parameter3) {
  // code to be executed
}

Arrays

Arrays are used to store multiple values within a variable.

var holidays = ["Easter", "Halloween", "Thanksgiving", "Christmas"];

Associative Arrays

Associative arrays are arrays with named indexes. However, JavaScript doesn't support arrays with named indexes. In JS arrays always use numbered indexes.

Example of an associative array:

var person = [];
person[0] = "Rebecca";
person[1] = "Jamie";
person[2] = "Christopher"; 
document.getElementById("aArray").innerHTML =
"The person with the index numbered 2 is " + person[2] + " and the length of the array is " + person.length;

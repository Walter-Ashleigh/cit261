The canvas element is used to draw graphics on a page via scripting. The element itself is only a container and in order for graphics to be displayed you must use a script to draw the graphics. There's several methods for drawing paths, boxes, circles, text and more. The text drawn can be animated or non-animated. The objects inside canvas can move. The element can respond to user actions such as hover, button clicks or mouse clicks. The canvas element must contain an id as well as a defined height and width. You may use multiple canvas elements on one page. You can also style it with a border. Canvas is a two dimensional grid and the upper left corner's coordinates are (0, 0). Some methods that can be easily used are moveTo(), lineTo(), beginPath() and arc(). The first two allow you to draw a line with a starting and end point. The second two allow you to draw a circle with a starting point and another to determine the size.

Your browser does not support the canvas tag.

HTML5 Tags for Media

Video

The video tag specifies movie clips or other types of video streams. The supported file formats are: MP4, WebM and OGG. MP4 is most widely used because it's supported by more browser types.

HTML:

<video width="300" controls>
<source src="../cit261/Storm_Warning.mp4" type="video/mp4">
Your browser does not support the video tag.
</video>

Audio

The audio tag defines sounds or other audio streams. There are three supported file formats: MP3, WAV and OGG. MP3 is supported on more browsers than the other two formats. If you place text between the opening and closing audio tags, that text will be displayed only if the the user opened the page on a browser that doesn't support the audio tag. This is the same for the video and canvas tags. You can use the controls attribute to add audio controls to play, pause, adjust volume, loop, etc. The audio tag defines the sound content and the source tag defines the resources for the element.

HTML:

<audio controls loop>
<source src="../cit261/Night_Sounds_Crickets.mp3" type="audio/mpeg">
Your browser does not support the audio element.
</audio>

Canvas

The canvas tag is used to draw graphics on a page via scripting. The element itself is only a container and in order for graphics to be displayed you must use a script to draw the graphics. There's several methods for drawing paths, boxes, circles, text and more. The text drawn can be animated or non-animated. The objects inside canvas can move. The element can respond to user actions such as hover, button clicks or mouse clicks.

Examples:

Your browser does not support the canvas element.

JavaScript:

// locate the canvas element
var canvas = document.getElementById("myCanvas");
// create a drawing object
// getContext() is a built in html object with methods and properties
// for drawing
var ctx = canvas.getContext("2d");
// begins path
ctx.beginPath();
// determines the size of the circle
ctx.arc(95,50,40,0,2*Math.PI);
//changes the color of the line
ctx.strokeStyle = 'red';
// a line to appear in canvas
ctx.stroke();


Your browser does not support the canvas element.

JavaScript:

var canvas = document.getElementById("myCanvas2");
var ctx = canvas.getContext("2d");
// the fillStyle property fills in the object
// can be a color, gradient or pattern
ctx.fillStyle = "#FF0000";
// fillRect(x, y, width, height) draws a rectangle filled with the 
// fillStyle
ctx.fillRect(50, 50, 100, 100);

CSS3 Animations

Animation Building Blocks

You need two key things when creating an animation in CSS.

• Keyframes - This defines the stages and styling.

Keyframes include:

- The name of the animation: For example, "Swing".

- The stages of the animation: Each stage is represented with a percentage. 0% is the start and 100% would be the end. Intermediate stages in between can also be specified.

- The CSS properties: These would be the CSS properties that are defined and assigned to each stage. For example, transform, opacity, etc...

• Animation Properties - This defines the keyframes to assign to a specific element, as well as how it should be animated. This makes your animation actually run.

The Animation Properties need to make it run are:

- animation-name: This is the name you define in the keyframes.

- animation-duration: In seconds or milliseconds you define how long the animation lasts.

Other animation properties that can be used but aren't necessary to make it run, are:

• animation-timing-function: defines a speed curve (i.e linear, ease, etc.)

• animation-delay: upon loading, it delays the start of the animation for the specified time.

• animation-direction: defines the direction of the animation after a cycle has concluded.

• animation-iteration-count: specifies the number of times the animation will occur.

• animation-fill-mode: specifies a style for the element once it's finished playing. (i.e. if a box moves from the right to the left, you can have it return to it's starting position or leave it where it ended.)

• animation-play-state: pause or play the animation.

Example

Keyframes:

@keyframes swing { 
  /* At each interval the element will be rotated
    to the specified degree. The starting point is 0%
    and ending is 100% */
    20% { transform: rotate(15deg); } 
    40% { transform: rotate(-10deg); } 
    60% { transform: rotate(5deg); } 
    80% { transform: rotate(-5deg); } 
    100% { transform: rotate(0deg); } 
  }

Animation Properties:

.swing { 
 /* this is default if not set */
 transform-origin: top center;
 /* specifies the keyframes named 'swing' */
 animation-name: swing; 
 /* lasts 8 seconds */
 animation-duration: 8s;
 /* animation plays three times */
 animation-iteration-count: 3;
 /* the keyframes alternate forward and backwards */
 animation-direction: alternate;
}
